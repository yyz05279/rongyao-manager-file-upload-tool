问题：为什么显示 "Tauri 环境未初始化" ？
答案：因为您使用了错误的命令！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

❌ 错误方式（您可能这样做了）
--------------------------------

cd tauri-app
npm run dev        ← 这个命令只启动前端！

结果：
✅ 前端界面能显示
❌ 但后端没有运行
❌ invoke() 函数不可用
❌ 点击登录报错："Tauri 环境未初始化"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 正确方式（请这样做）
------------------------

cd tauri-app
npm run tauri:dev  ← 这个命令启动完整应用！

或使用完整路径：

cd "/Users/yyz/Desktop/熔盐管理文件上传工具/tauri-app"
npm run tauri:dev

结果：
✅ 启动 Rust 后端
✅ 启动 Vite 前端
✅ 打开应用窗口
✅ invoke() 正常工作
✅ 登录功能正常

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
